# Recipe Manager Web Application 🍴

A full-stack web application to manage, create, and share recipes. Users can register, log in, and manage their recipes with a secure JWT authentication system. Recipes can be created, updated, deleted, and searched by name.

---

## 📝 Features
- **User Authentication**: Register and log in with secure JWT tokens.
- **Recipe Management**: Create, update, and delete recipes with name, ingredients, and instructions.
- **Search Recipes**: Find recipes by name with real-time search.
- **Responsive Design**: Mobile-friendly UI for an optimal experience on all devices.
- **Secure**: JWT authentication ensures only authorized users can create, edit, or delete recipes.

---

## 🛠️ Tech Stack

**Frontend**:
- HTML5
- CSS3
- Vanilla JavaScript

**Backend**:
- Node.js
- Express

**Database**:
- MySQL

**Authentication**:
- JWT (JSON Web Token)
